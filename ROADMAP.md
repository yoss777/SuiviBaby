# SuiviBaby - Roadmap

## Version actuelle : 2.3 (MVP)

## Backlog - Idées futures

### Sommeil
- [ ] **Bruits blancs / Sons d'ambiance** - Intégrer des sons apaisants (bruit blanc, pluie, berceuse) pour aider bébé à s'endormir
  - Option légère : liens vers playlists externes (Spotify, YouTube)
  - Option complète : lecture audio intégrée avec timer auto-stop
  - Complexité : élevée (lecture arrière-plan, gestion audio, batterie)

### Photos / Moments
- [ ] **Filtres photo** - Ajouter des filtres vintage/doux pour les photos de bébé
- [ ] **Collages automatiques** - Générer des collages mensuels/annuels
- [ ] **Export album** - Créer un PDF/livre photo des meilleurs moments

### Statistiques
- [ ] **Graphiques avancés** - Courbes de croissance OMS, tendances sommeil/alimentation
- [ ] **Comparaison périodes** - Comparer semaine actuelle vs précédente
- [ ] **Insights IA** - Suggestions basées sur les patterns détectés

### Partage famille
- [ ] **Notifications push partagées** - Alerter les co-parents d'un nouvel événement
- [ ] **Commentaires sur événements** - Permettre aux proches de commenter
- [ ] **Rôles et permissions** - Lecture seule vs édition pour les invités

### Santé
- [ ] **Rappels vaccins** - Calendrier vaccinal avec notifications
- [ ] **Suivi médecin** - Historique des consultations et ordonnances
- [ ] **Export médical** - PDF formaté pour le pédiatre

### Technique
- [ ] **Mode hors-ligne** - Synchronisation différée quand pas de réseau
- [ ] **Widget iOS/Android** - Accès rapide depuis l'écran d'accueil
- [ ] **Apple Watch / Wear OS** - Saisie rapide depuis la montre
- [ ] **Siri / Google Assistant** - Commandes vocales natives

## Notes

- Prioriser les fonctionnalités à forte valeur utilisateur avec faible complexité
- Valider avec les retours utilisateurs avant développement
- Garder l'app simple et intuitive (ne pas surcharger)
